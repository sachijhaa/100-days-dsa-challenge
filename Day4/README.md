# Day 04 - Maximum Subarray

## Problem

LeetCode 53 - Maximum Subarray

## Approach

Used Kadane's Algorithm to maintain the maximum subarray sum ending at each index. At every step, either start a new subarray from the current element or extend the existing subarray, whichever gives a larger sum.

## Time Complexity

O(n)

## Space Complexity

O(1)

## Learned

* Kadane's Algorithm
* Dynamic Programming optimization
* Handling negative numbers efficiently
* Finding maximum subarray sum in linear time

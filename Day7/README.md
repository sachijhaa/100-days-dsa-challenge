# Day 07 - Rotate Image

## Problem

LeetCode 48 - Rotate Image

## Approach

Rotated the matrix in-place by first transposing it and then reversing each row. Transposition swaps rows and columns, while reversing each row produces a 90-degree clockwise rotation.

## Time Complexity

O(n²)

## Space Complexity

O(1)

## Learned

* Matrix transposition
* In-place matrix manipulation
* Combining transpose and reverse operations
* Efficient 90-degree clockwise rotation without extra space

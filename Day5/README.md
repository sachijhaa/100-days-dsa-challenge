# Day 05 - Sort Colors

## Problem
LeetCode 75 - Sort Colors

## Approach
Used the Dutch National Flag Algorithm with three pointers (low, mid, and high) to sort the array in a single traversal without using any extra space.

## Time Complexity
O(n)

## Space Complexity
O(1)

## Learned
- Dutch National Flag Algorithm
- Three-pointer technique
- In-place array sorting
- Efficient partitioning of elements

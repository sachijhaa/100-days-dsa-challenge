# Day 02 - Pascal's Triangle

## Problem
LeetCode 118 - Pascal's Triangle

## Approach
Generated each row using the combinatorial formula for binomial coefficients instead of calculating values from previous rows repeatedly.

## Time Complexity
O(n²)

## Space Complexity
O(n²)

## Learned
- Pascal's Triangle pattern
- Binomial coefficient (nCr) concept
- Efficient row generation using mathematics
- Avoiding redundant computations
